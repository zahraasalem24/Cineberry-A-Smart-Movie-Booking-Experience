# Cineberry-A-Smart-Movie-Booking-Experience

**Background**

The Cineberry Movie Booking system is a Python-Based desktop website developed to allows users to browse 
movies, view showtimes, book, Tickets, and manage their cinema experience conveniently. It provides an easy way for users to view current and upcoming movies, and allow users to check showtimes and seat availability. Additionally, it enables secure online booking and payment for tickets, and improve the overall convenience of the cinema-going experience.

Originally developed in (2025-2026 / 1st Term) as part of an academic course, and uploaded to GitHub later for documentation and portfolio purposes.

**System Features**

1. User registration and login system.
2. Viewing available movies with details (title, genre, duration, description).
3. Showing cinema halls and available showtimes.
4. Seat selection interface.
5. Booking and confirming movie tickets.
6. Payment simulation.
