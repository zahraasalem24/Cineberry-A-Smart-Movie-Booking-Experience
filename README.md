# Cineberry-A-Smart-Movie-Booking-Experience

Project Overview:

This project developed a web application that allows users to browse 
movies, view showtimes, book, Tickets, and manage their cinema experience 
conveniently.

Objectives:

• Provide an easy way for users to view current and upcoming movies.

• Allow users to check showtimes and seat availability.

• Enable secure online booking and payment for tickets.

• Improve the overall convenience of the cinema-going experience.


Scope of the Project:

1. User registration and login system.
2. Viewing available movies with details (title, genre, duration, description).
3. Showing cinema halls and available showtimes.
4. Seat selection interface.
5. Booking and confirming movie tickets.
6. Payment simulation.
